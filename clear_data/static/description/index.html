Business Testing Data Clear

You can define default model group list by yourself to help your work.

Key features
  • Easily clear any odoo object data what you want.
  • Define own model group list, clear the special model group data what you want.
    In this way, you can manage object data according to your own different management levels.
  • Can reset model id to 1 when model data was clear.
  • Support odoo version 12 11 10 .
Module Detials

Operation steps

Define model groups to manage object data according to different management levels

Reset model sequence

Clear business data

Notes

  1. Do not use the module in a production environment

    As a data cleansing tool, data can be clear easily, but restore dificultly. So do not use it in a production environment.

  2. Do not have to clean up the basic object data

    Do not easily clean the product, product classification and other basic object data. The module will clean up the default data of the base object, causing some functions of the module to not work normally. If you have to clean up, cleaning up the object, you need to re-upgrade the module where the object to be cleaned up to restore the default data.
