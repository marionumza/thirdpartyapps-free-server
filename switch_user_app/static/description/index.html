User Switch

  • Features
    • Allow User Switch To Other User Without Logout.
    • After User Switch Module Will Replace Current User Session By New User Session.
    • Allow Change Configuration To Enable Switch User Or Disable It.
    • Allow Change Configuration To Make Switch User Without Password.
    • Allow Change Configuration To Make Switch User By Password.
    • Add Two Security Groups:
    • Settings : For Change Configuration
    • Switch User : To Enable Switch User , Without This Group, User Will Not able To Swith User

Two Group In User .

Switch User Menu In Settings

With Password Option In Setting , If Check Will Show Password Input To Specify By User

After Enable User Switch In Setting , Switch User Menu Will Appeare In

Switch User Dialog For Choose User,No Password Required If Configuration Without Password

Switch User Dialog For Choose User With Password Required If Configuration Choose With Password
